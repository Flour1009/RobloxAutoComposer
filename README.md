# RobloxAutoComposer
Скрипт генерирует случайные мелодии для Roblox Piano в стилях pop, ballad и ambient, автоматически выбирает тональность, темп и аккорды, а затем воспроизводит ноты через клавиатуру. Поддерживает горячие клавиши запуска/остановки, ограничение полифонии и настройку скорости проигрывания.
