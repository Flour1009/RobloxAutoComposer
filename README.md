# README для Roblox Piano
# скрипт генерации музыки
*Подробное описание скрипта, его возможностей и порядка использования*

## Назначение скрипта
Этот скрипт предназначен для автоматической генерации простой музыки для Roblox Piano с последующим воспроизведением нот через клавиатуру. Он не читает готовые MIDI-файлы, а сам создает музыкальный материал на основе выбранного стиля, тональности, темпа и набора внутренних правил.
Скрипт полезен в тех случаях, когда нужно быстро получить случайную мелодию, аккомпанемент и ритмический рисунок без ручного написания нот. После генерации программа переводит ноты в нажатия клавиш, которые соответствуют раскладке виртуального пианино.
## Что делает скрипт
Скрипт случайным образом или по заданным параметрам выбирает стиль композиции, тональность, лад, темп и длину произведения в тактах.
Далее он строит последовательность аккордов по заранее подготовленным гармоническим схемам, создает мелодическую линию, добавляет бас и арпеджио, а затем превращает все это в последовательность нажатий клавиш для Roblox Piano.
В результате пользователь получает автоматически сгенерированную музыкальную мини-композицию, которую можно проиграть прямо в игре или в другом приложении, где используется такая же клавиатурная раскладка.
## Основной функционал
* автоматическая генерация мелодии на основе музыкальных шаблонов;
*	поддержка нескольких стилей: pop, ballad и ambient;
*	автоматический или ручной выбор мажора и минора;
*	автоматический или ручной выбор тональности;
*	выбор количества тактов будущей композиции;
*	автоматический подбор гармонической последовательности;
*	создание мелодии, басовой линии и арпеджио;
*	ограничение полифонии, чтобы не зажимать слишком много клавиш одновременно;
*	горячие клавиши для запуска и остановки воспроизведения;
*	настройка скорости проигрывания и длины удержания нот;
*	отображение служебной информации: seed, темп, тональность, секции и диапазон нот.
## Как устроен скрипт
Внутри программа делится на несколько логических частей. Сначала определяются музыкальные данные: ноты, интервалы, гаммы, стили, аккордовые последовательности и соответствие нот клавишам. Затем создаются функции генерации мелодии и аккомпанемента. После этого все ноты преобразуются в события вида «нажать клавишу» и «отпустить клавишу».
На последнем этапе скрипт ожидает нажатие клавиши старта, активирует окно Roblox, а затем воспроизводит сгенерированную последовательность с учетом времени, скорости и ограничений по числу одновременно удерживаемых нот.
## Используемые библиотеки
Для работы используются стандартные библиотеки Python argparse, threading, random, time и dataclasses, а также сторонние библиотеки pydirectinput, pygetwindow и pynput.
pydirectinput отвечает за отправку нажатий клавиш, pygetwindow помогает найти и активировать окно Roblox, а pynput используется для отслеживания горячих клавиш запуска и остановки.
Основные зависимости:
Библиотека	Назначение	Обязательность
argparse	Обработка аргументов командной строки	Да
threading	Отдельный поток для клавиши остановки	Да
random	Случайная генерация музыки	Да
time	Работа со временем воспроизведения	Да
dataclasses	Структура BeatNote	Да
pydirectinput	Эмуляция нажатий клавиш	Да
pygetwindow	Поиск и активация окна Roblox	Да
pynput	Отслеживание горячих клавиш	Да
## Как подготовиться к запуску
Перед запуском должен быть установлен Python и все необходимые библиотеки. Скрипт ориентирован в первую очередь на запуск в Windows, потому что эмуляция клавиш и работа с окном Roblox обычно используются именно там.
Также желательно заранее открыть Roblox Piano, переключить раскладку клавиатуры на английскую и убедиться, что нужное окно находится в доступе для активации.
Пример установки зависимостей:
```pip install pydirectinput pygetwindow pynput```
## Как запустить скрипт
Скрипт запускается из командной строки. После старта он создает музыкальную последовательность, выводит информацию о будущей композиции и ждет нажатия горячей клавиши запуска.
Когда пользователь нажимает клавишу старта, программа начинает последовательно отправлять нажатия в Roblox Piano. Для принудительной остановки используется отдельная клавиша остановки.
Базовый пример запуска:
```python script.py --style pop --scale auto --key auto --bars 16```
## Аргументы командной строки
Аргумент	Описание	Пример
--start-key	Клавиша начала воспроизведения	f8
--stop-key	Клавиша остановки	f9
--speed	Множитель скорости. Больше 1 — медленнее, меньше 1 — быстрее	1.5
--style	Музыкальный стиль: pop, ballad, ambient	pop
--scale	Лад: auto, major, minor	major
--key	Тональность: auto, C, D#, F, A и т.д.	C
--bars	Количество тактов	16
--bpm	Темп. 0 означает автоматический выбор	0
--seed	Зерно генерации. -1 означает случайное значение	-1
--base-midi	Базовая MIDI-нота начала диапазона	36
--max-poly	Максимум одновременно удерживаемых нот	3
--min-hold	Минимальная длина удержания ноты	0.05
--max-hold	Максимальная длина удержания ноты	0.55
## Порядок использования
1.	Установить Python и необходимые библиотеки.
2.	Сохранить код скрипта в файл, например script.py.
3.	Открыть Roblox Piano или другое совместимое окно с нужной раскладкой клавиш.
4.	Переключить раскладку клавиатуры на английскую.
5.	Запустить скрипт с нужными параметрами через командную строку.
6.	Посмотреть информацию о сгенерированной композиции в консоли.
7.	Нажать клавишу старта, указанную в параметре --start-key.
8.	При необходимости остановить воспроизведение клавишей из параметра --stop-key.
## Что выводится в консоль
После генерации скрипт показывает seed, выбранный стиль, тональность, темп, количество тактов, диапазон доступных нот и структуру секций. Это удобно для повторного воспроизведения удачной генерации.
Если сохранить seed и те же параметры запуска, композицию можно получить повторно или очень близко к исходному варианту.
## Пример готового запуска
```python script.py --style ballad --scale minor --key A --bars 16 --bpm 84 --start-key f8 --stop-key f9```
В этом примере будет создана композиция в стиле ballad, в ладу minor, в тональности A, длиной 16 тактов и с темпом 84 BPM. После запуска программа будет ждать клавишу F8 для старта и F9 для остановки.
## Ограничения и важные замечания
*	скрипт зависит от правильной раскладки клавиатуры и соответствия клавиш виртуальному пианино;
*	если окно Roblox не найдено или не активируется, воспроизведение может уйти не в то окно;
*	слишком высокая скорость или слишком длинная композиция могут ухудшить точность воспроизведения;
*	если часть нот выходит за доступный диапазон клавиш, они не будут сыграны;
*	эмуляция клавиш может требовать запуска от имени администратора в некоторых системах.
## Для чего подходит этот скрипт
*	для автоматического создания простых музыкальных заготовок;
*	для развлечения и импровизации в Roblox Piano;
*	для тестирования клавиатурной раскладки пианино;
*	для быстрого получения случайных мелодий без ручного набора нот;
*	для экспериментов с генеративной музыкой на Python.

*Краткий итог: Этот скрипт представляет собой генератор простой процедурной музыки с автоматическим проигрыванием через клавиатуру. Он объединяет выбор тональности, аккордов, ритма, мелодии и сопровождения в одном файле и позволяет быстро получить готовую музыкальную последовательность для Roblox Piano.*
